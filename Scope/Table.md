| Scope Type     | Where Declared           | Accessible Where?         |
| -------------- | ------------------------ | ------------------------- |
| Global Scope   | Outside all functions    | Everywhere                |
| Function Scope | Inside function          | Only inside that function |
| Block Scope    | Inside {} with let/const | Only inside that block    |


a → Global → Accessible everywhere
b → Function → Accessible inside function
c → Block → Accessible only inside if block
